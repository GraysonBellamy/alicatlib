# `alicatlib.registry`

Primer-verified registries of gases, units, statistics, and
loop-control variables. Generated at build time from
[`codes.json`](../../src/alicatlib/registry/data/codes.json) via
[`scripts/gen_codes.py`](../../scripts/gen_codes.py); a CI idempotency
check guards against drift. See [Design](../design.md) §5.3.

::: alicatlib.registry
